# DoNa 앱 서비스 가이드

DoNa는 AI 기반 데이트 코스 추천 앱. 서울 주요 지역의 큐레이션된 코스를 추천하고, 방문 후 추억을 기록할 수 있다.

---

## 앱 구조 (하단 탭 5개)

```
홈 | 코스 | 탐색 | AI | 마이페이지
```

---

## 유저 여정

### 신규 유저
```
설치 → 스플래시(4초) → 온보딩(취향 설정) → 로그인/회원가입 → 홈
```

### 재방문 유저
```
앱 열기 → 홈 → AI 추천 or 코스 탐색 → 코스 방문 → 추억 기록
```

---

## 온보딩 (최초 1회)

가입 후 4단계 취향 설정. 이 데이터로 홈 추천 코스 개인화.

| 단계 | 질문 | 선택지 |
|------|------|--------|
| 1 | 분위기 | 힐링 / 힙스터 / 로맨틱 / 액티비티 |
| 2 | 가치관 | 분위기·뷰 중시 / 맛 중시 |
| 3 | 동행자 | 연인 / 친구 / 혼자 / 소개팅 |
| 4 | 선호 지역 | 성수·건대 / 홍대·연남 / 종로·북촌 / 을지로 / 강남·압구정 / 한남·이태원 / 잠실·송파 / 여의도·영등포 (복수 선택) |

마이페이지 → 프로필에서 언제든 재설정 가능.

---

## 탭별 기능

### 홈

| 섹션 | 설명 | 조건 |
|------|------|------|
| 활성 코스 배너 | 현재 진행 중인 코스 표시, 탭하면 코스 상세로 이동 | 로그인 + 진행 중인 코스 있을 때 |
| 오늘의 선택 | 온보딩 취향 기반 개인화 추천 코스 | 항상 표시 |
| AI 맞춤 추천 CTA | AI 탭으로 바로 이동 버튼 | 로그인 시 |
| 나만의 추억 | 내가 기록한 추억 사진 갤러리 | 로그인 + 추억 있을 때 |

---

### 코스 탭

- 히어로 슬라이더: 에디터 추천 코스
- 컨셉 필터: 감성데이트 / 맛집탐방 / 힐링 / 이색데이트 / 야경 / 인생샷 / 술자리 / 공연·전시 등 11개
- 무한 스크롤 목록
- 각 코스 카드에서 바로 찜하기 가능

---

### 탐색 탭 (네이버 지도)

- 지도 위에 주변 코스/장소 마커 표시
- 마커 탭 → 코스 미리보기
- 드래그 바텀시트로 목록 확인
- 내 위치 버튼으로 현재 위치로 이동
- 초기 중심: 서울시청 (자동 현위치 탐색은 버튼으로만 동작)

---

### AI 탭

**하루 1회 코스 추천** (구독 플랜에 따라 횟수 다름).

채팅 FLOW:
```
오늘의 목적 선택
├── 기념일 → 기념일 종류 선택
└── 무난 / 감성 / 활동 / 트렌디
        ↓
동반자 선택 (연인 / 썸 상대 / 소개팅 상대 / 친구 / 혼자)
        ↓
지역 선택 (문래·영등포 / 합정·용산 / 안국·서촌 / 을지로 / 여의도)
        ↓
코스 추천 결과 → courses/{id} 상세 페이지로 이동
```

- 추천 결과는 마이페이지 → 기록 탭 → **AI추천저장**에 자동 저장
- 채팅 대화 내용은 저장 안 됨 (매번 새로 시작)
- 한도 초과 시 구독 업그레이드 안내 시트 표시

#### 구독 플랜별 AI 추천 횟수

| 플랜 | 횟수 |
|------|------|
| FREE | 제한 있음 |
| BASIC | 제한 있음 (FREE보다 많음) |
| PREMIUM | 무제한 |

---

### 코스 상세 (`courses/{id}`)

| 기능 | 설명 |
|------|------|
| 지도 보기 | 코스 장소들 마커 + 폴리라인 표시 |
| 장소 순환 | 코스 내 장소들 순서대로 탐색 |
| 찜하기 | 마이페이지 → 기록 → 찜한 코스에 저장 |
| 공유 | 네이티브 공유 시트 (링크: dona.io.kr/courses/{id}) |
| 리뷰 | 별점 + 코멘트 작성 (코스당 1개, 재작성 시 업데이트) |
| 코스 시작 | 추억 기록 화면으로 이동 |

---

### 추억 기록 (`courses/{id}/start`)

코스 방문 후 기록 남기기.

| 항목 | 제한 |
|------|------|
| 사진 | 최대 10장, S3 업로드 |
| 별점 | 1~5점 |
| 설명 | 자유 텍스트 |
| 태그 | 선택형 |

기록 완료 → 마이페이지 → 발자취 → 나만의 추억 갤러리에 추가됨.

---

### 마이페이지

4개 탭 구조:

#### 프로필 탭
- 닉네임·프로필 사진 수정
- 취향 온보딩 재설정
- 알림 설정
- 카카오 채널 문의
- 로그아웃 / 회원탈퇴
- 구독 플랜 표시

#### 발자취 탭
- 활동 캘린더 (코스 방문 날짜 표시)
- 나만의 추억 갤러리 (기록한 사진들)

#### 기록 탭

| 서브탭 | 내용 |
|--------|------|
| 찜한 코스 | 찜하기 누른 코스 목록 |
| AI추천저장 | AI가 추천해준 코스 목록 (자동 저장) |
| 완료 코스 | 추억 기록 완료한 코스 목록 |
| 케이스파일 | 이스케이프 게임 완료 기록 |

#### 활동 탭
- 뱃지: 특정 조건 달성 시 획득
- 보상 내역
- 결제 내역 (구독 결제 이력)

---

### 이스케이프 게임

별도 콘텐츠. 코스와 연계된 미션/스토리 형태의 게임.
- 목록 → 상세 → 플레이
- 완료 시 케이스파일에 기록

---

### 오늘 뭐하지 (`nearby`)

코스가 아닌 단일 장소 탐색.
- 키워드 검색
- 무한 스크롤
- 찜하기

## 구독 / 결제

- 인앱 결제: RevenueCat 연동 (App Store / Google Play)
- 구독 관리: 마이페이지 → 프로필 → 구독 플랜
- 결제 이력: 마이페이지 → 활동 → 결제 내역

---

## 알림 (푸시)

- 마케팅 동의한 유저에게만 발송
- 설정: 마이페이지 → 프로필 → 알림 설정
- 수신 거부 시 서비스 알림(코스 시작 등)은 유지, 마케팅 알림만 중단

---

## 공유 / 딥링크

- 코스 공유 링크: `dona.io.kr/courses/{id}` → 앱에서 열면 네이티브 코스 상세로 이동
- 커스텀 스킴: `duna://`
