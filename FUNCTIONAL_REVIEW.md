## 유저 여정 (핵심 플로우)

### 신규 유저
```
앱 설치 → 스플래시(4초) → 온보딩(취향 설정 4단계) → 로그인/회원가입 → 홈
```

### AI 코스 추천 (핵심 가치)
```
AI 탭 → FLOW 채팅 (목적/동반자/예산 등 선택)
→ 코스 추천 결과 → courses/{id} 상세 보기
→ 찜하기 / AI추천저장 (마이페이지 기록 탭에 저장)
→ 코스 시작 → 추억 기록 (사진+별점+태그)
→ 마이페이지 발자취에 추억 누적
```

> AI 추천은 하루 1회 제한. 채팅 대화는 저장 안 됨(매번 새로 시작), 추천된 코스 결과만 저장.

### 코스 탐색
```
코스 탭 → 컨셉 필터 선택 → 코스 목록 → 코스 상세
→ 지도에서 장소 순환 → 찜하기 / 바로 시작
```

### 주변 탐색
```
탐색 탭 → 네이버 지도 (초기: 서울시청) → 내 위치 버튼 → 주변 장소 마커
→ 마커 클릭 → 코스 상세
```

## 웹 (Next.js / dona.io.kr)

웹과 앱은 동일한 API 서버 공유. 웹은 SEO, 공유 링크 미리보기, Magazine 등 담당.

| 경로 | 설명 |
|------|------|
| `/` | 홈 |
| `/courses` | 코스 목록 |
| `/courses/{id}` | 코스 상세 (공유 링크 OG 포함) |
| `/share/course/{shareId}` | 코스 공유 프리뷰 |
| `/mypage` | 마이페이지 |
| `/admin` | 관리자 페이지 |
| `/privacy`, `/terms` | 법적 문서 |
